# Test Task

## App has 2 screens:
* **Task List** 
* **Weather** 

## Technology

 - The app is written with **Dart Flutter**
 - State Management: **BloC**
 - Design: **Material Design 3**
 - API: **OpenWeather**
 - Work with Network: **Dio**
